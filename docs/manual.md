## 1. 설치

### 설치가 필요 없는 단일 exe
1. [Releases 페이지](https://github.com/vavibolivia/YtConverter/releases/latest) 이동
2. `YtConverter.App.exe` 다운로드 (~72 MB)
3. 원하는 폴더(예: `D:\Apps\YtConverter\`) 에 저장
4. **더블클릭**으로 실행

### 최초 실행 주의
- 최초 1회만 **FFmpeg(~100 MB) 를 자동 다운로드**합니다 (약 15~30초)
- 저장 위치: `%LOCALAPPDATA%\YtConverter\ffmpeg\` (사용자별 1회)
- 이후 실행은 즉시 시작됩니다

### 시스템 요구사항
- **Windows 10 (21H2 이상) 또는 Windows 11**
- **x64 CPU**
- 인터넷 연결
- 디스크 여유 공간 **최소 500 MB** (FFmpeg + 변환 버퍼)

## 2. 기본 사용법

### MP3 로 변환 (음원 추출)
1. 앱 실행
2. YouTube 영상 URL 복사 (브라우저 주소창)
3. URL 창 클릭 후 **Ctrl+V** (또는 **붙여넣기** 버튼)
4. **MP3 (오디오)** 라디오 선택
5. **변환 시작** 클릭
6. 진행률 100% 될 때까지 대기 (보통 10초 내외)
7. **폴더 열기** 로 결과 확인

### MP4 로 변환 (영상 저장)
위와 동일하되, 4단계에서 **MP4 (영상)** 선택.

## 3. UI 요소 설명

```
┌─────────────────────────────────────────────────────┐
│  YouTube Converter                      [_][□][X]  │
├─────────────────────────────────────────────────────┤
│ ①URL       [                          ] [붙여넣기] │
│ ②포맷      (•) MP3 (오디오)  ( ) MP4 (영상)        │
│ ③저장 폴더 [...................] [찾아보기][열기]  │
│ ④[변환 시작] [취소]                                 │
│ ⑤진행 상황                                          │
│    영상 제목 : "...."                               │
│    상태      : 다운로드 중 62%                      │
│    [███████░░░░░]                                    │
│ ⑥로그                                                │
│   2026-04-18 22:00  INFO  앱 시작                   │
└─────────────────────────────────────────────────────┘
```

| 번호 | 요소 | 설명 |
| --- | --- | --- |
| ① | URL 입력 + 붙여넣기 | 유튜브 주소 입력. `watch?v=...`, `youtu.be/...`, `&list=...` 모두 가능 |
| ② | 포맷 라디오 | MP3 는 오디오만, MP4 는 영상+오디오 |
| ③ | 저장 폴더 + 찾아보기/열기 | 기본: `%USERPROFILE%\Music\YtConverter` |
| ④ | 변환 시작 / 취소 | 변환 중에는 Start 비활성, Cancel 활성 |
| ⑤ | 진행 상황 | 제목, 상태(해석/다운로드/변환/완료), 진행률 바 |
| ⑥ | 로그 | 최근 500 줄 유지, 자동 스크롤 |

## 4. 단축키 & 팁

| 작업 | 방법 |
| --- | --- |
| URL 붙여넣기 | URL 창 포커스 후 **Ctrl+V**, 또는 **붙여넣기** 버튼 |
| 변환 취소 | **취소** 버튼 (진행 중일 때만 활성) |
| 폴더 열기 | **폴더 열기** 버튼 — 탐색기에서 결과 폴더 오픈 |
| 로그 복사 | 로그 아이템 선택 → **Ctrl+C** |

**팁**
- **동일 URL 중복 변환 시** 파일명 뒤에 `(1)`, `(2)` 자동 부여 → 덮어쓰기 방지
- **긴 제목** (120자 초과) 은 자동으로 잘립니다
- **금지 문자** (`/ \ : * ? " < > |`) 는 자동으로 `_` 로 치환

## 5. 저장 위치

| 종류 | 경로 |
| --- | --- |
| 변환 결과 기본 | `%USERPROFILE%\Music\YtConverter\` |
| FFmpeg 캐시 | `%LOCALAPPDATA%\YtConverter\ffmpeg\` |
| 앱 로그 | `%LOCALAPPDATA%\YtConverter\logs\yt-YYYYMMDD.log` |

`%LOCALAPPDATA%` 는 보통 `C:\Users\<이름>\AppData\Local\` 입니다.

## 6. 로그 & 문제 해결

### 로그 보기
- **앱 내 로그창** — 최근 활동 요약
- **파일 로그** — 탐색기 주소창에 `%LOCALAPPDATA%\YtConverter\logs` 입력 후 최신 `.log` 파일 열기

### 자주 발생하는 문제

| 증상 | 해결 |
| --- | --- |
| 최초 실행 시 변환이 멈춘 듯 | FFmpeg 다운로드 중입니다. 15~30초 대기 |
| "네트워크 오류" | 인터넷 연결, 방화벽/프록시 확인 |
| "재생 불가 영상" | 연령제한/지역제한 영상은 변환 불가 |
| "삭제되었거나 비공개" | URL 재확인 |
| 변환이 실제로 진행 안 됨 | 앱 종료 후 `%LOCALAPPDATA%\YtConverter\ffmpeg\` 를 지우고 재실행 |
| 저장 폴더에 파일이 없음 | 로그창 확인 — "변환 완료" 라인의 경로 참고 |
| Windows Defender 경고 | 서명되지 않은 exe 라서 SmartScreen 경고 가능 → "추가 정보" → "실행" |

## 7. 에러 메시지 사전

| 메시지 | 원인 | 조치 |
| --- | --- | --- |
| `라이브 스트림은 변환할 수 없습니다` | 라이브 URL | 방송 종료 후 VOD 로 재시도 |
| `삭제되었거나 비공개 영상입니다` | 비공개/삭제 | URL 확인 |
| `재생 불가 영상입니다 (연령/지역 제한)` | age-gate/region-lock | 로그인 크롬에서 재생 가능한지 확인 |
| `네트워크 오류. 다시 시도하세요` | 타임아웃/DNS | 연결 확인 후 재시도 |
| `저장 공간이 부족하거나 파일을 쓸 수 없습니다` | 디스크 부족 또는 쓰기 권한 없음 | 다른 폴더 선택 |
| `오디오 스트림을 찾지 못했습니다` | 매우 드문 케이스 (댄스 라이브 등) | 다른 영상으로 테스트, 이슈 리포트 |

## 8. 고급 사용법

### 다른 저장 위치 사용
- **찾아보기** → 네트워크 드라이브, USB, NAS 등 지정 가능
- 기본값은 윈도우 사용자 프로필 하위

### 동시 여러 개 변환
- 현재 UI 에선 1건씩만 가능 (차후 큐 지원 예정)
- CLI 스트레스 러너로 테스트 가능:
  ```bash
  dotnet run --project src/YtConverter.Tests -- --stress
  ```

### 변환 결과 검증
- 로그창에서 `변환 완료: ...` 경로 확인
- 파일을 Windows Media Player / VLC 에서 재생
- 개발자라면 `ffprobe` 로 코덱/비트레이트 검증:
  ```
  ffprobe -v quiet -show_streams -show_format "output.mp3"
  ```

## 9. 개인정보 & 네트워크

- 앱은 **YouTube 및 gyan.dev (FFmpeg)** 외에는 외부 통신을 하지 않습니다
- **텔레메트리/광고/추적 없음**
- 로그·FFmpeg 캐시는 **로컬에만** 저장됩니다
- URL 이나 결과 파일이 외부로 전송되지 않습니다

---

## 10. 제거 / 초기화

### 앱 제거
1. `YtConverter.App.exe` 파일 삭제
2. (선택) 캐시·로그도 지우려면: 탐색기에 `%LOCALAPPDATA%\YtConverter` 입력 후 폴더 삭제
3. (선택) 변환 결과: `%USERPROFILE%\Music\YtConverter\`

### FFmpeg 재설치
- `%LOCALAPPDATA%\YtConverter\ffmpeg\` 폴더 삭제 후 앱 재실행
